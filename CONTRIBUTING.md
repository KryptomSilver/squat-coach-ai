# Contributing

Gracias por contribuir a Squat Coach AI.

## Reglas basicas

- No eliminar la autoria original del proyecto.
- Mantener la licencia MIT en redistribuciones y derivados.
- Documentar cambios relevantes en codigo, README o notas del PR.
- Evitar sobreingenieria: priorizar claridad y simplicidad para el MVP.

## Buenas practicas

- Hacer cambios pequenos y enfocados.
- Mantener compatibilidad con el pipeline local existente.
- No introducir dependencias pesadas sin justificacion clara.
